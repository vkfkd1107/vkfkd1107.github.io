# 프로세스와 스레드

> **프로세스**와 **스레드**는 운영체제에서 작업을 실행하는 기본 단위입니다. 두 개념의 차이점과 특징을 알아보겠습니다.

## 🔄 프로세스 (Process)

프로세스는 **실행 중인 프로그램의 인스턴스**입니다.

### 📋 프로세스의 특징

- **독립적인 메모리 공간** 보유
- **프로세스 ID (PID)** 로 식별
- **생성/종료 비용이 높음**
- **안전성**: 한 프로세스의 오류가 다른 프로세스에 영향 X
