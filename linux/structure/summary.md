# 리눅스 구조 완전 가이드

인프라를 구축하고 운영하는 개발자가 꼭 알아야 하는 리눅스 시스템의 핵심 구조와 동작 원리입니다.

## 목차

- 리눅스란 무엇인가?
- 커널과 사용자 공간
- 리눅스 배포판의 차이점

- 주요 디렉토리 설명 (/etc, /var, /usr, /home 등)
- 파일 권한과 소유권
- 심볼릭 링크와 하드 링크

- 부팅 로그 확인 방법
- 서비스 자동 시작 설정

### 4. 프로세스와 데몬

- 방화벽 개념
- 네트워크 설정 파일들

### 6. 웹서버 동작 원리
- HTTP 프로토콜 기초
- 웹서버의 역할 (Apache vs Nginx)
- 정적 파일 vs 동적 처리
- 리버스 프록시 개념

### 7. 서비스 관리
- systemd 서비스 관리
- 서비스 시작/중지/재시작
- 서비스 상태 확인
- 로그 관리

### 8. 보안 기초

- SSH 설정과 보안
- 파일 권한 설정

### 9. 모니터링과 로그
- 시스템 리소스 모니터링
- 로그 파일 위치와 의미
- 로그 분석 방법
